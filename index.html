Atividade Avaliativa

Esta é uma atividade avaliativa da disciplina de Programação Web. Ela será utilizada para avaliar o seu conhecimento e habilidades na criação de páginas web utilizando HTML e CSS com a ajuda do framework Tailwind CSS.

Sobre a disciplina

A disciplina de Programação Web tem como objetivo apresentar os principais conceitos e técnicas utilizadas no desenvolvimento de páginas web. Durante o curso, você irá aprender a criar páginas web utilizando HTML, CSS e JavaScript, bem como utilizar frameworks e bibliotecas populares, como o Tailwind CSS, para facilitar o desenvolvimento.
